# ReqForge Documentation Site

Built with [Astro](https://astro.build).

```bash
npm install
npm run dev
```

## Structure

```
src/pages/
├── index.astro          # Landing page
├── docs.astro           # Docs index
└── docs/
    └── getting-started.astro
```

## Build

```bash
npm run build  # outputs to dist/
```
